2. DESCRIERE ȘI STRUCTURA PROIECTULUI
------------------------------------------------------------------------
Proiectul reprezintă o aplicație realizată în Unreal Engine versiunea 5.7.4. 

Repository-ul conține exclusiv codul sursă și asset-urile esențiale necesare reconstrucției proiectului, fiind curățat de fișiere binare compilate inutile.

Structura principală a directoarelor încărcate:
* /Config  - Fișierele de configurare ale proiectului (.ini).
* /Content - Toate asset-urile aplicației (hărți, blueprint-uri, materiale, modele texturi).
* Survival.uproject - Fișierul principal de proiect Unreal Engine.


3. PAȘII DE COMPILARE AI APLICAȚIEI
------------------------------------------------------------------------
Pentru a genera fișierele binare și structura completă a proiectului local, urmați acești pași:

Pasul 1: Clonați repository-ul sau descărcați arhiva proiectului.
Pasul 2: Asigurați-vă că aveți instalat Unreal Engine (versiunea 5.7.4 sau o versiune mai recenta) și Visual Studio.
Pasul 3: Dați click dreapta pe fișierul "Survival.uproject" și selectați opțiunea "Generate Visual Studio project files".
Pasul 4: Deschideți fișierul "Survival.uproject" în Unreal Editor. 
Pasul 5: La promptul care vă întreabă dacă doriți să reconstruiți modulele lipsă ("Missing Survival Modules - Would you like to rebuild them now?"), selectați "Yes". Editorul va compila automat codul/fișierele binare necesare.


4. PAȘII DE INSTALARE ȘI LANSARE A APLICAȚIEI
------------------------------------------------------------------------
-----Rularea direct din Editor 
1. După deschiderea proiectului în Unreal Editor (conform pașilor de compilare de mai sus), așteptați să se încarce toate shaderele.
2. În bara de instrumente de sus (Toolbar), apăsați pe butonul "Play" (sau folosiți scurtătura Alt + P) pentru a lansa aplicația în modul Play In Editor (PIE).
